<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your bot up and running in minutes with these simple steps.
  </p>
  
  <Steps>
    <Step title="Clone and Install">
      Clone the repository and install dependencies with npm.
      
      ```bash
      git clone https://github.com/ryu51474/preguntalealprofe.git
      cd preguntalealprofe
      npm install
      ```
    </Step>
    
    <Step title="Configure Environment">
      Create a `.env` file with your bot credentials and API keys.
      
      ```env
      BOT_TOKEN=your_telegram_bot_token
      numeroAdmin=569xxxxxxxx
      implementacionApiGoogle=your_google_apps_script_code
      ```
      
      <Note>
        Get your Telegram bot token from [BotFather](https://t.me/botfather) and set up your Google Apps Script for data management.
      </Note>
    </Step>
    
    <Step title="Launch the Bot">
      Start the application and scan the QR code for WhatsApp authentication.
      
      ```bash
      npm start
      ```
      
      The bot will generate a QR code in your terminal for WhatsApp Web authentication.
    </Step>
  </Steps>
</div>
